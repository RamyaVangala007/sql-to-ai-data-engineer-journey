## Daily Progress

### Day 1 — Python Environment + First ETL Pipeline
**What I built:**
- Set up a professional Python environment (venv, VS Code, Git)
- Built my first ETL pipeline using Pandas
- Read a CSV, transformed data with filter/groupby/sort
- Saved output as Parquet format
- Wrapped everything in a function with try/except error handling

**Key concepts learned:**
- ETL = Extract → Transform → Load
- Pandas maps directly to SQL (WHERE, GROUP BY, ORDER BY)
- Parquet is better than CSV for data engineering
- Virtual environments keep projects isolated

**Tech stack used:**
- Python 3.13
- Pandas, PyArrow, Jupyter
- Git + GitHub
